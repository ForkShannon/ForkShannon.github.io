Markdown语法

发表于2021-09-07|更新于2021-09-18
|阅读量:

1.什么是Markdown?

​ markdown是一种轻量级标记语言，它允许使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML或HTML，由于Markdown的轻量化、易读易写特性，并且对于图片和图表、数学公式都有很好的支持，目前许多网站都广泛使用Markdown来撰写说明文件。

2.块级元素

1. 标题

Markdown支持两种形式的标题，[Setext][1]和[atx][2]。
Setext样式的标题使用等号来表示一级标题，使用连字符表示二级标题，其中=或-的长度没有限制可以是任意长度。例如：

这是一级标题

这是二级标题

Atx样式的标题每行开头使用1-6个#号，分别对应1-6级标题，#号与标题内容之间用一个空格隔开。例如：

这是一级标题

这是二级标题

这是三级标题

这是四级标题

这是五级标题
这是六级标题

2.块引用

​ Markdown使用>字符作为块引用,同样的>和引用内容之间用一个空格隔开，通常来说最好对引用文本采取强制换行并在每一行行首放一个>。例如：

那一年我二十一岁，在我一生的黄金时代，我有好多奢望。我想爱，想吃，还想在一瞬间变成天
上半明半暗的云。 ——《黄金时代》王小波

后来我才知道，生活就是个缓慢受锤的过程，人一天天老下去，奢望也一天天消逝，最后变得像
挨了锤的牛一样。可是我过二十一岁生日时没有预见到这一点。我觉得自己会永远生猛下去，什
么也锤不了我。 ——《黄金时代》王小波

Markdown中也可以简便地只在每一个需要强制换行的段落的首行前面加上一个>：

那一年我二十一岁，在我一生的黄金时代，我有好多奢望。我想爱，想吃，还想在一瞬间变成天上半明半暗的云。 ——《黄金时代》王小波

后来我才知道，生活就是个缓慢受锤的过程，人一天天老下去，奢望也一天天消逝，最后变得像挨了锤的牛一样。可是我过二十一岁生日时没有预见到这一点。我觉得自己会永远生猛下去，什么也锤不了我。 ——《黄金时代》王小波

块引用可以嵌套，在块引用中包含块引用，只需要添加额外层级的>即可：

这是第一层块引用

这是第二层块引用

回到第一层块引用

同样的在块引用中也可以包含其他的Markdonw元素，包括标题、列表和代码块等：

这是二级标题

  1. 这是第一个列表项
  2. 这是第二个列表项

3.列表

​ Markdown支持有序列表和无序列表，无序列表使用*号、+号和-号作为列标记，这三个符号是等价的可以互换。

  • Red
  • Green
  • Blue
  • Red
  • Green
  • Blue
  • Red
  • Green
  • Blue

有序列表使用数字加.号构成，不能忘记的是在Markdown元素与内容之间一定要用空格隔开：

  1. C/C++
  2. JAVA
  3. Python

4.水平线

​ 如果一行中只有三个以上的连字符，星号或者下划线则会在该位置生成一个<hr/>标签。星号和连字符之间允许有空格，但是要注意连字符必须与上一行文字之间空一行。

3.内联元素

1.链接

​ Markdown支持内联链接，链接的界定符是[中括号]。内联链接只需要在[中括号]中写上链接的文本内容，[中括号]后面紧接一对圆括号，圆括号中放所需要的URL。例如：

这是一个内联链接案例。

这个链接没有title属性。

2.强调

​ Markdown使用星号*和下划线_作为强调标记，使用*或者_包裹的文本将会用HTML<em>标签包裹；双*或者_将会使用HTML<strong>标签包裹。例如：

single asterisks
single underscores
double asterisks
double underscores

3.代码

Use the printf() function.
